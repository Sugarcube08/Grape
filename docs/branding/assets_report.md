# Grape Brand Assets Report

This report catalogs the verified branding source assets located in the staging directory `grape/` and describes their target usage in the Android mobile application.

| Asset | Format | Resolution | Target Usage |
| :--- | :--- | :--- | :--- |
| `logo-1024.png` | PNG | 1024 × 1024 px | Combined application logo, action bar logo, and About page branding. |
| `foreground.svg` | SVG | 1024 × 1024 px (embedded image) | Source material for the adaptive icon foreground layer. |
| `background.svg` | SVG | 1024 × 1024 px (viewBox, 1020 × 1020 embedded image at x=3, y=2) | Source material for the adaptive icon background layer. |
| `monochrome.svg` | SVG | 1024 × 1024 px (viewBox, 924 × 973 embedded image at x=50, y=26) | Source material for the adaptive themed icon and the white notification icon. |
| `favicon-512.png` | PNG | 512 × 512 px | High-resolution favicon for browser and web interfaces. |
| `splash-logo.png` | PNG | 1024 × 1024 px | Central brand logo displayed during the Android 12+ splash screen animation. |
| `wordmark.svg` | SVG | 1024 × 1024 px | Vector wordmark representation for high-fidelity branding. |
